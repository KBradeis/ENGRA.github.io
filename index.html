Project 1:Building with Prusa

This was a cool starter project. I enjoyed relearning how to use 3D printers and learning about using 2 colors on one print. One thing I’m still curious about is how the 3D printer figures out how much space it leaves between the nozzle and print plate. I hope learning how to calibrate this isn’t too much of a hassle.

I learned that depending on how you upload the file the pre-set conditions/settings could be lost. I have never run into this problem so me and Ian had to troubleshoot and brainstorm on what we think the problem is. I had inserted the print by going to file and importing it into Prusa. If I had dragged it into Prusa instead I would have never run into this problem.

This is what it should look like if you drag the file into Prusa. Now the Keychain will come out with two different colors rather than one. I want to learn how to manually put multiple colors on any prints that I want to make.

This is what my stress test looks like. I was able to make this a small print. Through this print I was able to learn how to use the tools on Prusa slicers. I was able to test every limit. This print might have gone through over extrusion making certain details look messy and hard to read.We can’t see any words or specific details on the piece I printed.

This was my open source assistive device. I chose this since I wanted a tool that could help people communicate. In the photo you can see a tool used for people who are visually impaired communicate with others through a tactile language. I think the documentation for this print was very detailed and easy to understand.I don’t think I can give back any immediate suggestions on what to improve.

Project 2: Using TinkerCad and Onshape

I didn’t know what to do or where to start so I decided to do something that would impact me positively. So I wrote things that would help make my room more organized. Starting with my everyday essentials, my phone, wallet, keys , and watch. I decided to look for something that could store them and manage the wires while I charge my phone at the same time.

This was what it looked like in TinkerCad and Prusa. I decided to add support to prevent anything from going wrong. Overall I liked how firm the structure is, I won’t have to worry about it breaking if it falls down. Working in TinkerCad was something I haven’t done in a while so I had to shake off some rust.

Here’s the first print draft. It needs to be bigger to fit in all my essentials. I think I could also improve the wire holder that is currently on this design. Overall I would rate it a 2 on a scale of 1 - 5 based on its utility.I like the first design.

In Onshape this was what I designed. I enjoyed relearning how to use all my tools. I think I might work on this design to make it more intune with whatever I want in my dorm room. My favorite part about this project is that I can do whatever I want as long as I can imagine it and customize it to my needs.

Reflection: Overall this was a cool project. I realised the problem wasn’t the prints at all. It seemed to be the size that I made them in the Prusca slicer. Which means both of my ideas could have worked if they had been made bigger before they were printed. Working on tinkercad is a bit harder when making things isn’t ideal. It is a bit clunky since I have to get shapes extremely close to each other to connect them yet it still might end up with spaces in between them making it hard to print then. While Onshape is a bit better it’s a bit more challenging to use since nothing is intuitive, everything has to be found out.
